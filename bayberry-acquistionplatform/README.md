
#采集流程

1. 编辑任务
2. 启动任务
3. 查看采集数据
4. 查看任务列表
5. 设置定时任务
6. 任务监控
7. 提供常用的模板




##任务规则

1.网址采集规则

设置一个或多个**起始网址**，在该网址中获取内容。

获取内容的方式有：**自动获取地址链接**， **手动设置规则获取**。

2.内容采集规则

先设置一个**标签**，比如“标题”“内容”（标签有CUID功能），

接着对该标签设置以下选项：

- 从源码中获取

  - 数据来源： 从默认页 -> 源码中

  - 提取方式
  
    - 前后截取
    
    - 正则截取
    
    - 正文截取
    
    - XPath提取
    
    - JSON提取

- 生成固定的格式

    生成数据的格式，有如下情况：

    - 固定的字符串
    - 系统时间
    - 随机字符串
    - 随机数字
    - 系统时间戳
    - 随机抽取信息

- 已有的标签组合


3.内容发布规则

内容发布规则有如下形式：

- WEB在线发布
- 导入到数据库
- 保存为本地文件
